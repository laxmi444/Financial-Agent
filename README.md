# Financial-Agent
An AI-driven assistant that provides real-time stock market news, financial insights, and investment updates, helping users make informed decisions efficiently.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/laxmi444/Financial-Agent
cd Financial-Agent
```

2. Install required packages:
```bash
pip install -r requirements.txt
```

3. Create a `.env` file in the root directory and add your API keys: (optional)
```bash
GROQ_API_KEY=your_groq_api_key_here
```

## Usage

1. Start the Streamlit app:
```bash
streamlit run app.py
```

2. Enter your Groq API key in the interface
3. Start asking questions related to stock!
